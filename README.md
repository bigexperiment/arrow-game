# ArrowX

## Recent Changes

- Reworked the game into an endless progression instead of manual level selection
- Added local progress saving for the current level
- Improved mobile layout so the board takes more of the screen
- Replaced random board generation with a guaranteed-solvable chain-based generator
- Added exact optimal-tap calculation for each generated board
- Reduced frozen arrows so they stay a light modifier, capped at 2 per board
- Kept debug/testing hooks in code, while commenting out the visible debug UI for now

## What It Is

ArrowX is a chain-reaction puzzle game built with React + Vite.

You tap an arrow, it fires in its direction, and any arrow it hits fires too. The goal is to clear the whole board before you run out of taps.

## How Boards Work

Boards are generated by building solution chains first, then turning those chains into playable layouts.

This gives us:

- guaranteed solvable boards
- exact optimal tap counts
- controlled difficulty ramp
- support for light modifiers like frozen arrows and decoys

## Difficulty Model

Difficulty increases through a mix of:

- more required chains
- longer or less obvious paths
- more decoys
- larger reading space
- lower room for mistakes

The game tracks an exact optimal solve count for each board, and we also use an internal difficulty score for testing.
